# data/

Хранилище состояния SEO-конвейера. НЕ удалять файлы вручную.

- `keywords.json` - реестр опубликованных ключей (защита от каннибализации). Пишет агент 7.
- `clusters.json` - семантические группы. Пишет агент 1.
- `research_cache.json` - кэш юр-фактов с источниками. Пишет агент 2 (append-only).
- `embeddings.sqlite` - векторы статей для проверки уникальности (sqlite-vec). Создаётся скриптом `tools/init_embeddings.py` на этапе 2.
- `queue.json` - очередь тем для `/batch-run`. Можно править вручную.
- `publication_log.json` - лог публикаций (создаётся агентом 7).
- `batch_log.json` - лог пакетных прогонов (создаётся `/batch-run`).

Файл `embeddings.sqlite` появится позже - когда добавим зависимость `sqlite-vec` и инициализационный скрипт.
