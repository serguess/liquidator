# Cheatsheet редактора (агент 6) - содержательная редактура и фактчек

Этот файл — выжимка из writer-cheatsheet.md, оставлена только та часть, которую делает редактор. Технические проверки (длина, заспамленность, AI-маркеры, density, сокращения, тире, кавычки, voice) централизованы в `tools/quality_gate.py` — их редактор НЕ выполняет руками. HTML-каркас, JSON-LD, CTA, дисклеймер, aside «Об авторе», шапку/подвал собирает `tools/inject_boilerplate.py` — их редактор НЕ пишет.

Полные правила в `writer-cheatsheet.md` / `anti-ai-style.md` / `yandex-quality.md` — открывать только при сомнениях по конкретному кейсу.

## ЧТО РЕДАКТОР ДЕЛАЕТ САМ (LLM лучше скрипта)

### 1. Фактчек юр-утверждений (двухпроходный)

**Проход 1 — соответствие.** Каждое утверждение в draft.md сверить с research.json. Несовпадение → откатить к `safe_phrasing` из research.

**Проход 2 — актуальность.** Для каждого факта с цифрой (порог, срок, ставка, пошлина, тариф) или ссылкой на пункт нормы:
- В research.json `last_verified` не старше 6 месяцев и `legislation_status: "active"`. Иначе → возврат на агента 2.
- Контекст применения нормы корректен (не путаются процедуры, стороны, категории долгов).

**Запрещено в финальном тексте:** факты без актуализации, нормы с устаревшей редакцией, числа без `last_verified`.

### 2. Закрытие интента в первом экране

Первые 3-5 предложений = прямой ответ на запрос (для featured-блоков Яндекса).

### 3. Запрещённые блоки (удалять если писатель вставил)

- «Шаги» / «Процесс шаг за шагом» / «Пошаговый план» / «Как это работает по шагам»
- «Шаблон заявления» / «Образец письма» / «Пример документа»
- «Проблемы и перспективы» / «Что нас ждёт в будущем» / «Заключение с оптимизмом»
- Раскрытие схем извлечения выгоды (контроль над управляющим, обход кредиторов)

Контрольный вопрос: «может ли читатель пройти процедуру без юриста по этой статье?» Если да — есть прикладные знания, убрать.

### 4. Лимит цитирования закона

Прямые цитаты норм закона (внутри `<blockquote>` или после «согласно ст. X») в сумме ≤ 600 знаков. Больше — заменить избыточные цитаты на пересказ. Снижает риск дубля с правовыми порталами.

### 5. Ссылки на закон

2-3 максимум в теле статьи (КонсультантПлюс / Гарант). Больше — удалить лишние гиперссылки, оставить текстовое упоминание «по 127-ФЗ».

### 6. Внутренние и внешние ссылки

- 2-4 внутренних на смежные статьи (из `outline.internal_links_targets`, проверить что target-slug есть в `data/published_index.json` со `source: "published"`).
- Минимум 2 внешних на КонсультантПлюс/Гарант из `research.key_sources`.
- Anchor — ключевая фраза статьи-цели, не «здесь» / «по ссылке».
- **Формат внутренних ссылок (строго):** `href="/articles/{category}/{slug}"` — c полным префиксом `/articles/{cat}/`, БЕЗ `.html` и БЕЗ trailing slash. Примеры: `/articles/fiz/bankrotstvo-pensionera` ✅, `/bankrotstvo-pensionera` ❌ (404), `/articles/fiz/bankrotstvo-pensionera.html` ❌ (301-редирект), `/articles/yur/kak-zakryt-ooo-s-dolgami/` ❌ (404). Канонический URL статьи виден в её `<link rel="canonical">` — внутренняя ссылка должна совпадать.
- Контроль: `tools/internal_links_check.py` встроен в `quality_gate` (шаг 5.5). Автофиксит `.html`, trailing slash и короткие ссылки `/{slug}`. Если slug отсутствует в `published_index.json` — hard-блок `broken_internal_links`.

### 7. Антипереоптимизация (Баден-Баден)

- Density главного ключа ≤ 2.5%.
- Точное вхождение главного ключа: 1 раз в title, 1 раз в H1, 1-2 раза в подзаголовках, 2-3 раза в теле. Остальное — части ключа / синонимы.
- Удалить «купить/заказать/оформить X в [город] недорого/срочно/быстро».

### 8. Категории — полные названия в видимом тексте

В крошках, тегах, BreadcrumbList JSON-LD:
- `fiz` → «Физические лица»
- `yur` → «Юридические лица»
- `vzysk` → «Взыскание задолженности»
- `news` → «Новости»

В URL/slug/`meta.category` — технические сокращения остаются.

### 9. Стилистическая редактура смысла

- Вертикальные списки с **жирным заголовком:** + двоеточие → нумерованный список или связные абзацы.
- Запрещённые формулировки CTA (если задал в `meta.cta_*_text`): «срочная помощь», «звоните», капс в HTML, восклицательные знаки.

### 10. Устойчивые юр-формулировки НЕ перефразировать

«Исполнительский иммунитет», «единственное пригодное для проживания жильё», «сведения внесены в ЕФРСБ», «неразрывно связанное с личностью кредитора», «арбитражный управляющий» (не «финансовый» / «конкурсный» вне цитаты).

### 11. Дефисные сцепки субъектов — расцеплять

Если писатель склеил две роли через дефис («кредитор-поставщик», «организация-должник», «Покупатель-банкрот», «кредитор-юридическое лицо», «должник-гражданин») — это ошибка правописания (text.ru метит красным), а не синоним. autofix расцепит автоматически, но если правишь смысл рядом — сразу оставляй ОДНО точное слово. Связка двух ролей дефисом запрещена.

## ЧТО РЕДАКТОР НЕ ДЕЛАЕТ

### Шаблонизатор `tools/inject_boilerplate.py` сделает за тебя

- HTML-каркас (`<!DOCTYPE>`, `<head>`, header, footer)
- Метатеги (title, description, og:*, canonical, robots)
- JSON-LD: Article, BreadcrumbList, FAQPage из `meta.faq`
- CTA-1 (после лида), CTA-2 (середина), CTA-3 (перед FAQ) — на местах placeholder-ов `<!--BP:CTA-TOP-->`, `<!--BP:CTA-MID-->`, `<!--BP:CTA-BOTTOM-->`
- Дисклеймер из `research.required_disclaimers[0]` или дефолт — на месте `<!--BP:DISCLAIMER-->`
- `<aside>` блок «Об авторе» (ООО ЛИКВИДАТОР)
- Breadcrumbs HTML
- `<section>` «Читайте также» + related-loader script
- Cache-bust версия CSS

Твоё дело — `body.html` (только содержание: lead + H2/H3 + FAQ + 4 placeholder-комментария) и `meta.json` со всеми полями.

### Quality_gate `tools/quality_gate.py` проверит за тебя (НЕ делай руками)

- Длина (hard 6000-8000, news 4500-6800)
- Заспамленность (top10/ngram3/lex.diversity)
- AI-маркеры (60+ regex, density ≤ 2/1000, 0 critical, ≤ 8 high)
- Сокращения юр-терминов (юрлиц/физлиц/дебиторк/финупр/маткапитал) — autofix
- Пробел после точки — autofix
- Длинные тире (—) — autofix
- Английские кавычки — autofix
- Де-ёфикация ё→е — autofix
- Дефисные сцепки субъектов (кредитор-поставщик, организация-должник) — autofix расцепляет
- Эмодзи (critical-маркер)
- Density главного ключа
- Голос «я» вместо «мы»
- Просторечия
- Шаблонные правовые фразы (anti-template)

При попытке сделать это руками — двойная работа, двойной возврат на писателя. Доверяй gate.
