# Anti-Spam Checklist - быстрая проверка перед сохранением (1 минута)

Открыть, когда draft.md готов. Запустить 4 grep'а, исправить проблемы, сохранить.

---

## ✓ Проверка 1: Дословные цитаты закона (30 сек)

**Поиск:**
```bash
grep -i "признаётся судом\|признание несостоятельности\|в соответствии с требованиями\|установленном порядке" drafts/{slug}/draft.md
```

**Если нашли:** переписать через **кейс** / **факт** / **активный залог**

❌ «несостоятельность гражданина признаётся судом»  
✅ «судья объявляет гражданина банкротом за 1-2 месяца»

---

## ✓ Проверка 2: Шаблонные фразы (30 сек)

**Поиск:**
```bash
grep -E "следует отметить|в целях|при наличии|согласно требованиям|в связи с тем|таким образом|более того" drafts/{slug}/draft.md
```

**Если нашли:** удалить или заменить на конкретику

❌ «в целях надлежащего исполнения»  
✅ «чтобы соблюсти закон» или просто удалить

---

## ✓ Проверка 3: Служебные слова (перенасыщение) (30 сек)

**Поиск:**
```bash
echo "Должник:" && grep -io "должник" drafts/{slug}/draft.md | wc -l
echo "Суд:" && grep -io "суд" drafts/{slug}/draft.md | wc -l
echo "Банкротство:" && grep -io "банкротство" drafts/{slug}/draft.md | wc -l
echo "Долг:" && grep -io "долг" drafts/{slug}/draft.md | wc -l
```

**Лимиты на 7000 знаков:**
- должник ≤ 10 раз
- суд ≤ 10 раз
- банкротство ≤ 12 раз (если главный ключ)
- долг ≤ 10 раз

**Если превышено:** применить лестницу синонимов из [anti-spam-red-flags.md](anti-spam-red-flags.md) категория 4

❌ «Должник подаёт заявление. Должник прикладывает документы.»  
✅ «Заявитель приносит документы. К ним - паспорт и СНИЛС.»

---

## ✓ Проверка 4: Синонимизатор (визуальная, 1 сек)

**Прочитать вслух последний абзац.**

**Если кажется странным** (Пример: «явление задолженности», «проведение процедуры») → это значит был синонимизатор.

**Исправить:** переписать предложение целиком, а не менять слова

❌ «При возникновении оснований подаётся обращение»  
✅ «Если долги выросли, идите в суд»

---

## Финальная команда

После исправлений:
```bash
python -m tools.quality_checks drafts/{slug}/draft.md && \
python -m tools.anti_template_check drafts/{slug}/draft.md
```

Если оба прошли — можно сохранять.
